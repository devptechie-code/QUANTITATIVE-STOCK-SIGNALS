# QUANTITATIVE-STOCK-SIGNALS 

A Python-based financial analysis tool that automates the process of trend detection. This script uses market data to help identify potential "Bullish" or "Bearish" signals for any stock ticker.

## Features
* **Real-time Data:** Pulls historical price data using the `yfinance` API.
* **Technical Indicators:** Calculates a 20-Day Simple Moving Average (SMA).
* **Signal Logic:** Detects price-to-average crossovers.
* **Visualization:** Generates a performance chart using `matplotlib`.

##  Installation

To run this project locally, you will need to install the following Python libraries:
"pip install yfinance matplotlib pandas"
